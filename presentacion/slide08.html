Patrón Strategy

Intercambio dinámico de proveedores LLM sin modificar el código

🎯 ¿Qué resuelve?

Permite cambiar el proveedor de LLM en runtime sin modificar ninguna clase de la aplicación.

📐 Estructura

  • LLMPort — Interfaz abstracta (contrato)
  • OllamaProvider — Implementación local
  • OpenAIProvider — Implementación cloud
  • GroqProvider — Implementación ultra-rápida

✅ Principios aplicados

  • OCP — Abierto a extensión
  • LSP — Subtipos intercambiables
  • DIP — Depende de abstracciones
  • SRP — Cada provider, una responsabilidad

⚡ En la práctica

# .env
LLM_PROVIDER=ollama
# Cambiar a:
LLM_PROVIDER=openai

¡Solo se cambia una variable de entorno!
