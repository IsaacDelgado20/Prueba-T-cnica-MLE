Buenas Prácticas Aplicadas

Conceptos de ingeniería de software implementados en cada capa

🧊

Frozen Dataclasses

Entidades inmutables con @dataclass(frozen=True). Previene mutaciones accidentales. Hashables.

🔒

SecretStr (Pydantic)

API keys no se imprimen en logs. SecretStr oculta datos sensibles automáticamente.

🕐

Timezone-Aware

Todos los timestamps usan datetime.now(UTC). Evita bugs de zona horaria en producción.

🔄

Retry + Backoff

Decorador @retry_with_backoff con jitter. Resiliencia ante fallos transitorios de red/LLM.

📋

Context Managers

Conexiones DB con with. Garantiza cierre limpio de recursos siempre.

🛡️

Parameterized Queries

Todas las queries usan ? placeholders. Cero riesgo de SQL injection.

🧵

Thread-Safe Singleton

Double-checked locking con threading.Lock(). Seguro para FastAPI async.

📝

Type Hints + Enums

Tipado estricto en todo el código. Enum para constantes seguras y autocompletables.

⚡

WAL Mode SQLite

Write-Ahead Logging para concurrencia. Lecturas no bloquean escrituras.

🏗️

Template Method

Base scraper con _do_scrape() abstracto. Scraping uniforme, adaptable por fuente.
