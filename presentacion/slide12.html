5 servicios orquestados con Docker Compose — Multi-stage builds y healthchecks

🏗️ Multi-Stage Build

# Stage 1: deps
FROM python:3.11-slim AS deps
COPY requirements.txt .
RUN pip install --no-cache-dir
# Stage 2: runtime
FROM python:3.11-slim AS runtime
COPY --from=deps /venv /venv

Imagen final ~60% más liviana

🏥 Healthchecks

  • Cada servicio tiene health endpoint
  • Intervalos configurados (30-60s)
  • depends_on: condition: service_healthy
  • Reinicio automático si falla

🔐 Seguridad Docker

  • Usuario no-root (USER appuser)
  • .dockerignore estricto
  • Variables via .env (no hardcoded)
  • Network aislada entre servicios
